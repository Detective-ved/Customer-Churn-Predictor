# Dataset Information

The dataset contains customer-related information used to predict whether a customer is likely to churn.

Typical attributes may include:

* Customer ID
* Gender
* Age
* Tenure
* Contract Type
* Monthly Charges
* Total Charges
* Internet Service
* Payment Method
* Churn Status

Target Variable:

* Churn

  * Yes = 1
  * No = 0

The dataset should be stored as:

customer_churn.csv
