# 📊 AI-Based Customer Churn Prediction System

An AI/ML project that predicts customer churn using Machine Learning algorithms such as Logistic Regression and Random Forest Classifier.

---

## 🚀 Features

* Data preprocessing and cleaning
* Label encoding for categorical data
* Feature scaling using StandardScaler
* Train-Test Split
* Logistic Regression implementation
* Random Forest Classifier implementation
* Model evaluation using:

  * Accuracy Score
  * Confusion Matrix
  * Classification Report
* Visualization using Seaborn and Matplotlib

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

---

## 📂 Project Workflow

Dataset → Data Cleaning → Encoding → Feature Scaling → Model Training → Prediction → Evaluation
